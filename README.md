# Local Secure VPN

A Python-based encrypted communication tunnel demonstrating VPN
fundamentals using AES encryption and secure socket communication.

## Features

- AES-256 encryption
- PBKDF2 key derivation
- Multi-client threaded server
- GUI and CLI clients
- Latency monitoring
- Config-driven setup
- Logging system

## Install

pip install -r requirements.txt

## Run Server

python server.py

## Run GUI Client

python gui_client.py
